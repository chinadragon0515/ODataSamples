RESTier Media Entity Support Sample
------------------

This sample illustrates how to support media type entity in RESTier.
User can practice with the sample like, (root URL is http://localhost:18384/api/mediaentity/)

  GET /Photos
  Get /Photos(key)/$value
  POST /Photos
  PATCH /Photos(key)
  POST /Photos(key)/$value


In this sample, the metadata of the photo is stored in database but photo file itself is stored in file system, user can customize the logic on where the data to be stored.

For a detailed description of RESTier, refer to document http://odata.github.io/RESTier/.
